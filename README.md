## Features ✨

The system is capable of:

- 🔍 **Searching restaurants**
  - Find restaurants based on user preferences and requirements.

- 📅 **Making reservations**
  - Book a table by providing reservation details such as date, time, number of guests, and customer information.

- ❌ **Canceling reservations**
  - Cancel existing reservations using reservation details.

- ❓ **Answering restaurant-related questions**
  - Provide information and answer user questions about restaurants, services, and policies.

## Overview

This project uses an AI agent approach to interpret user requests, select appropriate tools, and perform restaurant management tasks automatically.
